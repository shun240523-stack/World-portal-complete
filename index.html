接続の見取り図

  • USSO → Symbolencyclopedia

    記号の標準化と記号の収集がつながっています。理念側が USSO、アーカイブ側が Symbolencyclopedia です。

  • CED → NODE LAB

    CED がエネルギー供給源、NODE LAB がその応用先。技術基盤から人間の意識転送へ進む流れがあります。

  • BIB → NODE LAB

    NODE LAB のデジタル化に対して、BIB は肉体と実存を守る対抗軸として配置できます。

  • onva → !anbyam

    国家そのものと、その国家を支える最適化言語が分かれています。世界設定の本体と文法書をセットで扱うのが自然です。

このポータルの使い方

  • 入口として使う

    新しい訪問者がどこから見ればいいか迷わないように、全体像を先に見せます。

  • 設定資料として使う

    作品ごとの立場、相互関係、世界観の役割を整理することで、説明文のブレを減らせます。

    この下にカードを増やすだけで、別プロジェクトも同じデザインで収容できます。
